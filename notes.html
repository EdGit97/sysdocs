This page describes the purpose of this site and a number of miscellaneous operations.

About This Site

The Windows System Documentation site was created in order to document the details of the configuration of this Windows laptop. Each page documents a specific area of the system configuration.

The Start Menu and Firefox Bookmarks pages do not use data directly from the Start Menu and bookmarks directories. The pages instead use offline data that is copied to this site by the generate\getShortcut.vbs utility and the Firefox Manage Bookmarks, Import and Backup tool. The reason for this is, that if the system needs to be rebuilt from scratch, then the generated offline files can be used as a source to rebuild the applicable parts of the system.

The Backup Operations page maintains a list of extermal backup media and the number of times each storage device has been used. For details see the Backup Operations Technical Summary.

The remaining pages consist of static instructions that, when followed, allow the referenced part of the system to be re-installed and configured.

Change Powershell to Constrained Language Mode:

  1. Start -> Windows Powershell -> Windows Powershell
  2. PS> $ExecutionContext.SessionState.LanguageMode
  3. If the response is "FullLanguage" continue with the steps below, otherwise close Powershell
  4. Control Panel -> System and Security -> System
  5. Advanced system settings
  6. Environment Variables...
  7. System variables, click New...
    1. Variable name: __PSLockDownPolicy
    2. Variable value: 4
    3. OK
    4. OK
    5. OK
  8. PS> $ExecutionContext.SessionState.LanguageMode
  9. Result should now be "ConstrainedLanguage".

Power Reset

  1. Turn off the computer.
  2. Disconnect the AC adapter.
  3. Remove any discs from the disc drive.
  4. Remove memory cards from the card reader slot.
  5. Disconnect all nonessential peripheral devices, such as printers, scanners, external hard drives, and USB flash drives.
  6. With the AC adapter unplugged, press and hold the power button for approximately 15 seconds.
  7. Reconnect the AC adapter, and then turn on the computer.

Passwords

Passwords are stored in C:\Backup\pwds.zip. The ZIP file contains one password protected file, pwds.txt. If the system needs to be rebuilt from scratch, pwds.zip is backed up, unencrypted, to Google Drive at My Drive/Backup/pwds.zip. The Google Drive password and the Local Backup encryption password are also stored on the Android phone in FastNotePad:Passwords
