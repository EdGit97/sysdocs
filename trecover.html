Windows 10 Media Creation Tool - Install

  1. Download the Windows Media Creation Tool
  2. Download Windows 10
  3. Download Now
  4. C:\Src\setups\MediaCreationTool_22H2.exe
  5. Run MediaCreationTool_22H2.exe
  6. Select: Create installation media (USB flash drive, DVD or ISO file) for another PC
  7. Select language, architecture and edition: Use the recommended option for this PC
  8. Next
  9. Plug in the flash drive that will be used for the recovery media.
  10. Choose which media to use: USB flash drive
  11. Next
  12. Select a USB flash drive: F: (USB DISK)
  13. Next
  14. Windows 10 will now download, verify then create the media.
  15. Your USB flash drive is ready: Finish
  16. Eject the flash drive.

HP Pavilion dm1 - Windows 10 - Flash Drive Start

  1. With power off, insert the W10R recovery drive into a USB-A slot.
  2. Power on
  3. Esc
  4. F9 - Boot Device Options
  5. Select: USB DISK 3.0
  6. Choose your keyboard layout: US
  7. Recover from a drive
  8. Select appropriate:
    1. Just remove my files
    2. Fully clean the drive
