Find Phone

  1. Open Google Find My Device
  2. Select "Motorola moto g pure"
  3. Device's location will appear on the map.

Windows/Android Interface Setup

Enable Developer Options on the Android device:

  1. Settings -> About phone -> Build number -> Tap the build number seven times until until you get a message "You are now a developer!"
  2. Settings -> System -> Developer options -> USB debugging: ON
  3. Connect the Android device to a USB port on this laptop.

On the Windows laptop:

  1. Download the Android Debug Bridge to C:\Src\setups.
  2. Open C:\Src\setups\platform-tools_r34.0.4-windows.zip and extract the platform-tools directory to C:\Src\CellPhone.
  3. See attached devices: C>adb devices -l
  4. Download aapt. This utility takes a package identifier and determines the common name of the associated application.
  5. In GitHub: ... -> Download -> C:\Src\CellPhone\platform-tools
  6. Copy the aapt app to the Android device: C>adb push aapt-arm-pie /data/local/tmp/aapt
  7. Make aapt an executable: C>adb shell chmod 0755 /data/local/tmp/aapt
  8. To download the application list to this site, see instructions at the top of the Android Application page.

It is also possible to use the Android Debug Bridge (adb) to access the command shell. C>adb shell
